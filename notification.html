Job Details: Junior Web Developer
Job Summary:

A Junior Web Developer helps create and maintain websites and web applications. They work closely with senior developers and designers to support website development, fix bugs, and improve user experience. This is a great entry-level position for fresh graduates or those with basic coding experience.

Job Responsibilities:
  • Write and update code using HTML, CSS, JavaScript, and PHP
  • Assist in developing user-friendly, mobile-responsive websites
  • Help connect websites to databases and APIs
  • Test website features and fix errors or bugs
  • Work with designers and senior developers to build new features
  • Maintain and update existing websites as needed
  • Use tools like GitHub for version control and collaboration
Qualifications:

Educational Background:

  • Graduate of BS in Information Technology, Computer Science, or related course
  • (Optional) College undergraduates or vocational graduates with strong skills can also apply

Technical Skills:

  • Basic knowledge of HTML, CSS, JavaScript
  • Experience or familiarity with PHP (core and/or with a framework like Laravel)
  • Understanding of how to use MySQL or other databases
  • Can use Git/GitHub for version control
  • Bonus: Knowledge of WordPress, Bootstrap, or basic back-end development

Soft Skills:

  • Willing to learn and grow
  • Can follow instructions and meet deadlines
  • Good communication and teamwork skills
  • Attention to detail and problem-solving mindset

Preferred (But Not Required):

  • Internship or freelance experience in web development
  • A simple portfolio or GitHub account with sample projects
  • Experience using website builders or CMS platforms like WordPress
Expected Salary Range (Philippines):
  • Entry Level: ₱15,000 – ₱25,000 per month
  • With 1–2 years of experience: ₱25,000 – ₱35,000 per month
  • Freelance/Project-Based: ₱5,000 – ₱15,000 per project (depends on complexity)
